🎈 Windborne Balloon & Hurricane Landfall Analysis

This project analyzes the proximity between Windborne Systems balloon deployments and recent Atlantic hurricane landfalls using data from NOAA’s HURDAT2 archive. The goal is to evaluate how balloon positioning correlates with hurricane intensity.

🧪 Methodology

  • Fetched live balloon coordinates from Windborne’s API
  • Parsed recent hurricane tracks (2020–2023) from NOAA’s HURDAT2 dataset
  • Matched balloon positions to hurricane positions based on proximity (1–7 degrees)
  • Trained a Random Forest model to predict hurricane wind speed from balloon proximity and altitude
